# KROPA - Crop Disease Expert System

## Overview
KROPA is an expert system designed to assist farmers in diagnosing crop diseases based on observed symptoms. The system analyzes user input and provides possible diseases, causes, and recommended treatments.

## Objectives
- Help farmers identify crop diseases early
- Provide accurate treatment recommendations
- Improve agricultural productivity

## System Components
1. **Knowledge Base**  
   Contains structured data about crops, diseases, symptoms, and treatments.
2. **Inference Engine**  
   Matches user symptoms with stored disease rules and determines diagnosis.
3. **User Interface**  
   Allows farmers to input symptoms and displays diagnosis results.

## How It Works
1. User selects crop
2. User inputs symptoms or uploads image
3. System processes input
4. Inference engine compares input with knowledge base
5. Best match is returned
6. System displays disease, cause, and treatment

## Features
- Symptom-based diagnosis
- Simple and intuitive interface
- Expandable knowledge base
- Optional image-based detection (future)

## Example Use Case
A farmer notices yellow leaves and brown spots. KROPA analyzes symptoms and returns Leaf Blight, caused by fungal infection, with treatment recommendations.
